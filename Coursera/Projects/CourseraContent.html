HTML stands for HyperText Markup Language.

HyperText

Basically just text that links to other text

Markup Language

Language wrapped around the content that tells the computer about the nature of the content.

HTML, CSS, JAVASCRIPT

Each file provide respectively the Structure, the Style and the Behavior(functions) of the website.
Notice: because of the nature of HTML, everything in it serves the purpose of structuring, inlcuding the the heading tags.

Bit of history

Real origin of the current HTML:5 :WHATWG:HTML + W3C:XML

Tags

  • The !DOCTYPE html at the top of the html file is a declaration exclusive to HTML5. Without it, the browser enter quirks mode.
  • Like the Doctype tag, the title tag is also essential.
  • The viewport is refering to the body tag.
  • Something to remember: Html renders from top to bottom, sequentially.
  • Meta tags communicate information to the browser.
  • Div is a block element, hence even if it is followed by a span element, the div will push the span to a new line. Also the reason why you can't have a div inside span. The same goes for p element...
  • Pragraph tag, header and heading tags aswell as footer tags are all block level elements. The main reason of specifying instead of using div everywhere is to structurize the codes. Those are called semantic elements

Lists

Ordered(ol) and unordered(ul) lists with list item(li). Within a list, there can only be list items. It is posible to have a list within a list item.

Character Entity

Serves the purpose of preventing redering issues, basically replacing a special character with some other one.

Characters to avoid: <, > and & as they can cause confusion when interpreted by the computer. Instead, use & lt , & gt and & amp(withouth the space inbetween) respectively to represent those characters. A special one to remember is the copyright symbol ©, denoted by & copy.

Let's say I want some words to always stay on the same line, replace the space between the desired words with entity & nbsp which stands for None Breaking Space

Links

The a tag creats a hyperlink(href = hypertext reference) to either an internal or external link. For external links, it's suggested to add a target=_blank inside the starting tag so when the link is clicked, it appears in a new window.

The a element is both inline and block at the same time. This allows the a tag to have a div within it, meaning one is not obliged to click on the text but rather in the surrounding region to access the link.

Another feature of this element is to navigate within the same page. First an Id must be assigned to the designated sections, then creat a list in the following format: <li> <a href="#designated section">#Designated Section</a></li> *Of course, it doesn't necessarily have to be a list, any blcok element would work.

Example:Back to top

This feature(fragment identifier) changes the URl, hence it could be used as a bookmark when sending to someone. It brings them right to the designated section.
